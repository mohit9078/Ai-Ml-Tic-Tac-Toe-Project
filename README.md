# Ai-Ml-Tic-Tac-Toe-Project
Tic Tac Toe (Python Console Game)  A simple command-line Tic Tac Toe game built in Python. The game allows two players (X and O) to play turn-by-turn, detects winning patterns, and displays the updated board after every move. This project demonstrates basic Python programming concepts such as lists, functions, conditional logic, and loops.
